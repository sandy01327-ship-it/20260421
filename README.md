# ZenSpace - 正念與感恩空間

ZenSpace 是一個簡潔且優雅的原生 Web 應用程式，旨在幫助使用者在繁忙的生活中找到平靜。它結合了科學的呼吸引導與心理學實踐的感恩日記，提供一個無壓力的冥想環境。

## 主要功能

- **🧘 正念呼吸引導**: 採用 4-4-6-2 呼吸法（吸氣、憋氣、呼氣、放鬆），配合沉浸式動態視覺引導。
- **✍️ 感恩日記**: 記錄生活中的美好瞬間，支援持久化儲存於瀏覽器 LocalStorage。
- **🎨 自然調性設計**: 採用 Natural Tones 設計語彙，以米色與橄欖綠營造寧靜感。
- **📱 響應式佈局**: 支援電腦與平板等多種視圖模式（包含併列顯示）。

## 技術棧

- **框架**: React 19
- **樣式**: Tailwind CSS 4
- **動畫**: Motion (Framer Motion)
- **圖標**: Lucide React
- **建構工具**: Vite

## 如何在本地運行

1. **複製專案**
   ```bash
   git clone <your-repository-url>
   cd zenspace-mindfulness
   ```

2. **安裝依賴**
   ```bash
   npm install
   ```

3. **啟動開發伺服器**
   ```bash
   npm run dev
   ```

4. **瀏覽頁面**
   打開瀏覽器訪問 `http://localhost:3000`

## 如何部署

該專案為純前端應用（SPA），您可以輕鬆部署至：

# ZenSpace - Mindfulness & Gratitude Space

ZenSpace is a clean and elegant web application designed to help users find peace in their busy lives. It combines scientific breathing guidance with psychological gratitude practices.

## Key Features

- **🧘 Guided Breathing**: 4-4-6-2 breathing technique with immersive visual guidance.
- **✍️ Gratitude Journal**: Record positive moments with local storage persistence.
- **🎨 Natural Tones Design**: Calming aesthetic using beige and olive palettes.
- **📱 Responsive Layout**: Multiple view modes including side-by-side display.

## Tech Stack

- React 19, Tailwind CSS 4, Motion, Lucide Icons, Vite.
